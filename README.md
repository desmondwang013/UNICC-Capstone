# UNICC AI Safety Lab – Project 1: Research and Platform Preparation

## Overview
This repository contains the Project 1 work for the UNICC AI Safety Lab capstone. The goal of Project 1 is to define the research foundation, system architecture, platform assumptions, and technical specifications needed to support a standalone SLM-based multi-module AI safety ensemble.

## Scope
Project 1 focuses on:
- analyzing the top three Fall 2025 AI safety solutions
- defining the multi-module system architecture
- specifying module responsibilities and orchestration logic
- documenting DGX cluster assumptions and deployment setup
- preparing environment and configuration requirements

## Repository Contents
- research and architecture documentation
- functional and technical specifications
-  deployment, environment, and configuration planning


## Role in Team Project
This work provides the blueprint for:
- Project 2: fine-tuning the SLM and implementing the council of experts
- Project 3: system integration, testing, and user experience
